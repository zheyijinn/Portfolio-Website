WHAT I OFFER.

I am a UI/UX designer & full-stack web developer . I create user-friendly and visually appealing interfaces, and I also build strong and scalable web applications. I combine design and technology to make sure each project looks great and works well. With attention to detail and a love for innovation, I am committed to delivering high-quality solutions that meet and exceed client expectations.

UI/UX Design

Using Figma, I create detailed wireframes, mockups, and interactive prototypes to plan and test designs. I also gather valuable feedback through user research and testing to continuously improve the design and ensure a smooth user experience.

Web Development

I specialize in creating websites using tools like PHP, Laravel, Bootstrap, Tailwind CSS, HTML, and JavaScript. My goal is to craft smooth, responsive designs while working closely with designers and back-end developers to bring everything together seamlessly.

Front-end Development

With a focus on the user experience, I use technologies like Bootstrap, Tailwind CSS, HTML, and JavaScript to build visually appealing and interactive interfaces. I ensure everything works smoothly with the back-end and aligns with the design vision.

Back-end Development

As a freelance Full-stack Web Developer, I specialize in small to medium projects, using PHP, MySQL, and Laravel to create robust back-end systems and manage databases, ensuring smooth functionality and performance.
